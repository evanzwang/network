                                            README
                                      This folder contains code used to generate quantitative results used in the study titled "Mapping Three-Dimensional Tumor Heterogeneity through Deep Learning Inference of Spatial Transcriptomics from Routine Histopathology: A Proof-of-Concept Comparative Study" at ML4H 2024. Additional code for experiments is being de-anonymized and will be added to this repository. Please direct questions to the corresponding author Zarif Azher (
                                        [zazher@caltech.edu](mailto:zazher@caltech.edu)
                                        ).
